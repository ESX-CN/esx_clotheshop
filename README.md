## 要求
- [esx_skin](https://github.com/ESX-CN/esx_clotheshop)

## 下载 & 安装

### 使用 Git
```
cd resources
git clone https://github.com/ESX-CN/esx_clotheshop [esx]/esx_clotheshop
```

### 手动
- 下载 https://github.com/ESX-CN/esx_clotheshop/archive/master.zip
- 解压至 `[esx]` 目录

## 安装
- 添加如下设置至 `server.cfg`:

```
start esx_clotheshop
```
